# 行为模型（CZ 风格的抽象）

这里的行为模型并不是要「还原某个人」，而是从公开可见的信息里抽象出几条简单的风格：

- 面对日常波动时保持冷静；
- 在环境开始发生变化时提前进入「留意」状态；
- 在风险增加、波动变大时变得更加谨慎，降低冲动；
- 在极端情况下，用最明显的方式提醒「现在不正常」。

## 输入指标

当前版本只使用两个非常朴素的指标：

- `price_change_percent`：24 小时涨跌幅；
- `volume`：24 小时成交量（目前只用于展示，不参与阈值判断）。

## 状态定义

- `CALM`
  - 条件：|change| < 1%
  - 表现：眼睛为柔和绿色，舵机处于中性位置；
- `ALERT`
  - 条件：1% ≤ |change| < 5%
  - 表现：眼睛变为橙色，尾巴略微抬起，头部轻微转动；
- `CAUTIOUS`
  - 条件：5% ≤ |change| < 10%
  - 表现：眼睛变为黄色，尾巴放低，头部呈观察姿态；
- `PANIC`
  - 条件：|change| ≥ 10%
  - 表现：眼睛变为红色，姿态更紧绷，用来可视化「极端情况」。

这个模型刻意保持简单，方便在嵌入式设备上调试和扩展。真正的交易 / 风险判断应该在完全不同的系统中完成，而不是交给一只玩具机器人猫。
