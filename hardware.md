# 硬件说明

本项目默认使用如下硬件（可按需要替换）：

- **主控**：ESP32 DevKitC 或其他带 Wi‑Fi 的 ESP32 开发板
- **舵机**：2× SG90 / MG90S（一个控制尾巴，一个控制头部）
- **眼睛**：2× 独立 RGB LED 或一小段 NeoPixel 灯条（2~4 颗）
- **电源**：5V 直流电源，或锂电 + 稳压模块
- **连接**：通过 2.4GHz Wi‑Fi 连接到本地 MQTT 服务器

基础接线要点：

- 舵机的电源使用 5V，地线与 ESP32 共地；
- 舵机信号线分别接到 GPIO 25（尾巴）和 GPIO 26（头部）；
- NeoPixel DIN 接到 GPIO 13，串联一个 300~500Ω 电阻更稳；
- 所有模块共地，避免噪声导致舵机抖动。

具体外壳 / 机械结构可根据自己的机器人猫本体自由改造，本仓库只关心电子部分和软件逻辑。
